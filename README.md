# recruit

## サイト概要
就活の企業管理、可視化。企業に求める条件を設定し、企業がそれを満たしているかを自動で計算し、志望度をパーセントでをします。


### サイトテーマ
企業管理をより効率的にすることを目標にしています。


### テーマを選んだ理由。
いろんな媒体での応募は企業管理大変だったので、一つにまとめるかつ、効率的に就活、転職活動を進められるようなアプリを作りたいと思ったため。

### ターゲットユーザ
就活、転職中の方

### 主な利用シーン
就活、転職中
