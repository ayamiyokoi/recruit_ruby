# recruit

## サイト概要
就活の企業管理、可視化


### サイトテーマ
企業管理をより効率的にすることを目標にしています。


### テーマを選んだ理由。
いろんな媒体での応募→企業管理大変。一つにまとめるかつ、効率的に。

### ターゲットユーザ
就活、転職中の方

### 主な利用シーン
就活、転職中
